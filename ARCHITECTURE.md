# ElderDocAI Architecture

## Objective
Provide an AI-powered document understanding system for elderly users.

## Pipeline

1. Upload document
2. Store document
3. Extract text
4. OCR if necessary
5. Classify document
6. Extract key information
7. Generate elderly-friendly summary
8. Generate embeddings
9. Answer questions
10. Convert responses to speech
